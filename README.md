## How to deploy on GitHub Pages

1. Create a new repository on GitHub, e.g. `tanishq-portfolio`
2. Upload `index.html` to the root of that repo
3. Go to **Settings → Pages**
4. Under **Source**, select the `main` branch and `/ (root)` folder → **Save**
5. GitHub will give you a live URL within a minute or two, in the form:
   `https://<your-github-username>.github.io/tanishq-portfolio/`

## Updating the site later

Just edit `index.html` directly (in GitHub's web editor or locally) and push/commit —
GitHub Pages redeploys automatically within a minute of any change to `main`.
